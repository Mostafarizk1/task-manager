# نظام إدارة المهام - دليل البدء السريع

## 🚀 البدء السريع

### تشغيل التطبيق

```bash
npm run dev
```

افتح المتصفح على: `http://localhost:3000`

### 🔐 بيانات الدخول

#### المدير (ADMIN)
- **اسم المستخدم:** `admin`
- **كلمة المرور:** `admin123`

#### الموظف (USER)
- **اسم المستخدم:** `user`
- **كلمة المرور:** `user123`

## 📋 الميزات الرئيسية

### ✅ نظام الصلاحيات (RBAC)

| الميزة | ADMIN | USER |
|--------|-------|------|
| عرض جميع المهام | ✓ | ✗ (مهامه فقط) |
| عرض المبالغ المالية | ✓ | ✗ |
| إدارة المصاريف النثرية | ✓ | ✗ |
| التقارير الشهرية | ✓ | ✗ |
| إضافة/تعديل مهامه | ✓ | ✓ |
| حذف المهام | ✓ | ✗ |

### 💰 العملات المدعومة
- 🇸🇦 **SAR** - ريال سعودي
- 🇺🇸 **USD** - دولار أمريكي
- 🇪🇬 **EGP** - جنيه مصري

### 📊 حالات المهمة
- **نشط** (ACTIVE) - المهمة قيد العمل
- **مراجعة** (REVIEW) - المهمة قيد المراجعة
- **مكتمل** (COMPLETED) - المهمة منتهية

## 🎯 الاستخدام

### للمدير (ADMIN)

#### 1️⃣ إدارة المهام
- عرض جميع المهام من جميع الموظفين
- إضافة مهام جديدة مع المعلومات المالية الكاملة
- تعديل وحذف أي مهمة
- تتبع الأرباح لكل مهمة

#### 2️⃣ المصاريف النثرية
- الوصول: `/admin/expenses`
- إضافة مصاريف (نت، سلف، فواتير، اشتراكات)
- تتبع المصاريف حسب العملة
- عرض إجمالي المصاريف لكل عملة

#### 3️⃣ التقارير الشهرية
- الوصول: `/admin/reports`
- اختيار الشهر والسنة
- عرض تقرير مفصل لكل عملة:
  - إجمالي الدخل
  - صافي ربح المهام
  - المصاريف النثرية
  - **صافي الربح النهائي**

### للموظف (USER)

#### 1️⃣ إدارة المهام الشخصية
- عرض مهامه فقط
- إضافة مهام جديدة (بدون معلومات مالية)
- تعديل مهامه
- تغيير حالة المهام (نشط → مراجعة → مكتمل)

---

## 🧮 الحسابات التلقائية

### صافي ربح المهمة
```
صافي الربح = السعر الإجمالي - أجر المتعاون
```

### صافي الربح النهائي (شهري)
```
صافي الربح النهائي = صافي ربح المهام - المصاريف النثرية
```

**مثال (بالريال السعودي):**
```
مهمة 1: 5000 - 2000 = 3000 ربح
مهمة 2: 3000 - 1000 = 2000 ربح
مهمة 3: 4000 - 1500 = 2500 ربح
─────────────────────────────
صافي ربح المهام: 7500 ر.س

المصاريف النثرية: 1500 ر.س
─────────────────────────────
صافي الربح النهائي: 6000 ر.س ✓
```
