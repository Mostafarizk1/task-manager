# 📱 إعداد تنبيهات Telegram

## الخطوة 1️⃣: إنشاء Telegram Bot

1. افتح Telegram على التليفون
2. ابحث عن: `@BotFather`
3. ابدأ محادثة واكتب: `/newbot`
4. اختار اسم للبوت (مثال: `Task Manager Notifications`)
5. اختار username للبوت (لازم ينتهي بـ `bot` - مثال: `mytaskmanager_bot`)
6. **احفظ الـ Token** اللي هيديهولك (شكله كده: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

---

## الخطوة 2️⃣: الحصول على Chat ID

1. ابحث عن البوت بتاعك في Telegram (بالـ username اللي اخترته)
2. اضغط **Start** أو ابعتله أي رسالة
3. افتح الرابط ده في المتصفح (استبدل `YOUR_BOT_TOKEN` بالـ token بتاعك):
   ```
   https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
   ```
4. هتلاقي في الرد حاجة اسمها `"chat":{"id":123456789}`
5. **احفظ الـ Chat ID** (الرقم ده)

---

## الخطوة 3️⃣: إضافة المتغيرات في Vercel

1. روح على Vercel Dashboard
2. اختار المشروع `task-manager`
3. روح **Settings** → **Environment Variables**
4. ضيف المتغيرات دي:

| Name | Value |
|------|-------|
| `TELEGRAM_BOT_TOKEN` | الـ Token من BotFather |
| `TELEGRAM_CHAT_ID` | الـ Chat ID اللي جبته |
| `CRON_SECRET` | أي كلمة سر قوية (مثال: `my-secret-cron-key-2024`) |

5. اضغط **Save**
6. اعمل **Redeploy** للمشروع

## 🔧 الأوقات المضبوطة حالياً:

في ملف `vercel.json`:
```json
"schedule": "0 7 * * 0-4"   // 9 صباحاً (الأحد-الخميس)
"schedule": "0 10 * * 0-4"  // 12 ظهراً (الأحد-الخميس)
"schedule": "0 17 * * 0-4"  // 7 مساءً (الأحد-الخميس)
```

**Cron Format:**
- `0 7 * * 0-4` = الساعة 7 صباحاً UTC (9 صباحاً مصر) من الأحد للخميس
- `0-4` = الأيام من الأحد (0) للخميس (4)
- الجمعة (5) والسبت (6) مش مضمنين
